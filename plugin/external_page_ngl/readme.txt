This plugin is used to show a additional button in the course with NGL information embedded.<br>
<br>
Configuration<br>
=============<br>
<br>
Go to Administration > Configuration settings > Plugins<br>
Select the 'External Page NGL' plugin. And enable<br>
<br>
Enable tool: Enable the plugin in the courses<br>
<br>
Button Name: Set a button name to show in the courses tool<br>
<br>
Login process: Set the URL to sign-in to NGL<br>
<br>
Image Path: Set the button image<br>
<br>
Installation Notes<br>
==================<br>
<br>
When you install the plugin, it creates an extra user field (called ewoorkbooklogin) to record the user name<br>
And add a button in the course page to access the established sign-in process<br>
<br>
When you clicks the button to sign-in to login in NGL. The plugin open a new window. So, the browser may display a warning.<br>
The browser must allow pop-up windows open, for the sign-in process<br>
<br>
Unistallation Notes<br>
===================<br>
<br>
When you install the plugin, it remove the extra user field (called ewoorkbooklogin)<br>
